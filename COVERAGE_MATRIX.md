# EIED Cross-Benchmark Coverage Matrix

This matrix shows how each benchmark adds a distinct evidence-integrity capability while using the shared **Claim → Evidence → Confidence → Decision** framework.

| # | Benchmark | Primary evidence challenge | Main decision boundary | Distinctive suite contribution |
|---:|---|---|---|---|
| 001 | Discharge Documentation Closure Eligibility Audit | Signed-but-conflicting records; late results; wrong laterality; missing follow-up proof; stale copied-forward text; resolved apparent holds. | Has the record earned clean/complete closure status? | Separates administrative completion from evidentiary closure. |
| 002 | Prior Authorization Evidence Review | Criteria windows; body-region mismatch; therapy history; medication safety; exceptions; stale scratchpads; later addenda. | Has the request earned submission, hold, or escalation? | Tests exact criteria application without inheriting queue status. |
| 003 | Clinical Trial Eligibility Validation | Retired consent; pre-consent procedures; out-of-window labs; exact washout timing; current ECOG; interpreter attestation; final imaging. | Has the candidate earned the right to proceed? | Tests protocol-specific eligibility, timing, and exception evidence. |
| 004 | Registry Enrollment Audit | Outside record matching; spouse-result mismatch; incomplete colonoscopy; scheduled vs completed; total vs segmental colectomy; stale hospice flags. | Has the patient earned inclusion, exclusion, suppression, or outreach? | Tests registry truth, identity, and completion-status integrity. |
| 005 | Medical Necessity Documentation Review - Post-Acute Services | Template language; service-specific criteria; skilled need; homebound status; wound and oxygen proof; copied forms; later supporting notes. | Has the service earned a medically necessary determination? | Tests whether post-acute documentation earns medical necessity. |
| 006 | Medical Necessity Documentation Review - Utilization Management | Wrong body region; stale therapy; prescribed vs completed treatment; symptom-duration rules; identity mismatch; urgent exceptions; later addenda. | Has medical necessity been established for this service and patient? | Tests patient-specific procedural necessity and exception pathways. |
| 007 | Quality Measure Inclusion Validation | Qualifying encounters; BP source rules; exclusion evidence; repeat readings; outside values; dates; stale status; denominator vs numerator conflation. | Has the patient earned inclusion, exclusion, compliance, or hold status? | Tests measure membership and performance as separate claims. |
| 008 | Medication Safety Verification | Stale labs; claims vs active therapy; critical late values; interacting prescriptions; cancelled procedures; overdose history; delabeling; outside monitoring. | Has the medication plan earned release or withholding? | Tests release decisions for high-risk medication evidence. |
| 009 | Case Closure Readiness Audit | Attempted vs completed communication; planned vs verified action; wrong-patient evidence; final vendor reports; privacy obligations; owner/due-date gaps. | Has the case earned true resolution and closure? | Tests whether attempted work has become verified resolution. |
| 010 | Clinical Coding Evidence Validation | Abnormal lab vs diagnosis; empiric treatment; draft vs final documentation; baseline comparison; wrong-patient attachments; expected postoperative support; unsigned queries. | Has the diagnosis or principal-diagnosis assignment earned final coding? | Tests whether diagnosis and sequencing claims earn final coding. |
| 011 | Longitudinal Evidence Reconciliation | Copied-forward diagnoses; repeated but unsupported causality; service discharge vs recovery; contextually unreliable measures; recurrence after closure; stale holds resolved by current evidence. | Does the longitudinal record support the conclusion? | Tests whether the full longitudinal story supports the durable conclusion. |
| 012 | Clinical Guideline Applicability Audit | Unconfirmed qualifying diagnosis; independent eligibility pathways; wrong guideline version; acute vs chronic abnormalities; excluded populations; disease eligibility vs medication-safety gates. | Does the evidence actually justify applying this rule? | Tests the right to invoke a rule, not merely the ability to quote it. |
